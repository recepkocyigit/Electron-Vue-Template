
# Electron-Vue Template

Bu proje electron.jd ve vue.js ile hazırlanmış olup masaüstü uygulama oluşturmanıza yarar.


## Yüklü Kütüphaneler

Proje arayüzünde vue.js kullanılmıştır. Görsel anlamda daha da zenginleştirmek için vuetify kütüphanesi ve icon için material design icons paketi eklenmiştir. Başka stil kütüphanesi kullanmak istiyorsanız `npm install package-name` şeklinde kurulum yapabilirsiniz. Kurulumu yaptıktan sonra `src/renderer.js` dosyasına import etmeniz gerekmektedir. Bu dosyaya import etmediğiniz takdirde her component için ayrıca çağırmanız gerekir.

  
## Kurulum
Kurulum temelde 2 aşamadan oluşup şablonu kolayca yükleyebilirsiniz.
Proje dosyasını indirdikten sonra "npm install" komutu ile gerekli paketleri yükleyebilirsiniz. Başka ekstra kurulum gerektirmez.

```bash
  npm install
```

Kurulum bittikten sonra "npm start" komutu ile çalıştırabilirsiniz. 

```bash
  npm start
  npm run start (opsiyonel)
```

# Electron-Vue Template

This project is built with **Electron.js** and **Vue.js**, enabling you to create desktop applications with ease.

## Features

- **Vue.js** for building the user interface  
- **Vuetify** for rich UI components  
- **Material Design Icons** for icon support  
- Easy integration of additional style libraries (`npm install package-name`)

> ⚠️ If you install a new style library, make sure to import it in `src/renderer.js`.  
> Otherwise, you will need to import it separately in each component.

## Installation

Clone or download the project, then install the required dependencies:

```bash
npm install  
```

After installation use "npm start" to start project

```bash
  npm start
  npm run start (opsiyonel)
```
